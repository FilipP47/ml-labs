# traffic-signs-gan

Overview
 - Task: train a generative model to produce traffic sign images and generate samples for evaluation (e.g., FID).

Data
 - Dataset of traffic sign images (size/format depends on the project setup).

Solution
 - Model: conditional cDCGAN with stabilizing techniques (spectral norm, label embeddings).
 - Generation: 1,000 samples were saved for evaluation.
