# ML/DL Projects — Overview

This repository contains a set of machine learning and deep learning projects and experiments. Each project is self-contained and includes its own `README.md` and one-file solution to the given task.

Topics: regression, classification (images and tabular), sequence modeling (RNN), and generative models (GAN).

Folders:

- `bikes-regression/` — regression task predicting bike rentals.
- `house-price-classification/` — apartment price classification.
- `img-classifier/` — image classification (50 classes).
- `rnn-composer-recognition/` — composer recognition from chord sequences.
- `traffic-signs-gan/` — traffic sign image generation (GAN).


Technologies (examples): PyTorch, torchvision, scikit-learn.
